# pervaxis-forge-launchpad

Frontend for Pervaxis Forge — Angular 18 admin UI.

## Day 1
Read `docs/FORGE_SOLUTION_STRUCTURE.md` and `docs/FORGE_BLUEPRINT_UI.md` before writing any code.

## Run locally
```bash
npm install
npx nx serve launchpad
```

## Mock API
Set `useMockApi: true` in `apps/launchpad/src/environments/environment.ts` for week 1.
Flip to `false` on May 10 to connect to real BFF API.
